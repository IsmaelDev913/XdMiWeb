Uso del botón "Kick"

Este botón se utiliza para eliminar a un usuario de forma temporal o permanente de un grupo o canal. Al usar esta función, el usuario ya no podrá participar hasta que se le vuelva a dar acceso.

Uso del botón "Elimina al usuario"

Esta opción permite remover a un usuario de la lista de miembros de forma definitiva, impidiendo que vuelva a unirse a menos que sea invitado nuevamente. Es una función de eliminación permanente.
